### Step 1
Open a web browser and go to the official Android Studio website:  
https://developer.android.com/studio  
Download and install Android Studio by following the on-screen instructions.

### Step 2
After installation, open Android Studio and complete the Setup Wizard by selecting the default settings and installing the required SDK components.

### Step 3
Download Flutter SDK from the official Flutter website:  
https://flutter.dev  
Extract the downloaded ZIP file to a suitable location on your system (for example: C:\flutter).

### Step 4
Add the Flutter SDK path to the system environment variables so that Flutter commands can be run from the command prompt or terminal.

### Step 5
Open Android Studio, go to Settings > Plugins, search for Flutter, install it, and restart Android Studio when prompted.

### Step 6
After restarting, click on New Flutter Project from the Android Studio welcome screen and select Flutter Application.

### Step 7
Provide the Flutter SDK path, enter the project name, choose a project location, and click Next.

### Step 8
Select Android as the target platform, click Finish, and wait for Android Studio to create the project.

### Step 9
Connect an Android emulator or physical device, click the Run button, and your first Flutter application will launch successfully.
