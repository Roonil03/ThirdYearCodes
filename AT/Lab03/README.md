# Lab Demo Programs:
1. Design a UI using Text, Image, and different types of Material Buttons (Elevated, Outlined, and Text Button).
2. Develop an application that uses selection widgets like Checkboxes and Radio Buttons to collect user preferences.
3. Design a complex layout using nested Rows and Columns with the Expanded widget to distribute space.
4. Display a scrollable list of items using the ListView.builder pattern.

# Lab Exercises:
1. **Exercise 1 (Profile Dashboard):** Create a User Profile page. Use a Stack widget to place a profile picture (CircleAvatar) partially over a header image. List professional details below using ListTile.
2. **Exercise 2 (Survey Form):** Create a feedback form for a movie. Use TextField for the movie name, RadioListTile for ratings (1-5 stars), and a Switch for "Recommend to others?".
3. **Exercise 3 (Responsive Grid):** Use GridView.count to display a 2x2 grid of categories (e.g., Electronics, Fashion, etc.) where each cell contains an Icon and Text centered in a Card.
