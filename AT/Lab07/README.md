# Lab Exercises
1. Profile Dashboard
    - Create a profile screen.
    - Use ImageBackground.
    - Use circular profile image.
    - Display professional details using FlatList.
    - Add Edit Profile button.
2. Movie Feedback Form
    - TextInput for movie name.
    - Radio buttons (1–5 stars).
    - Switch for "Recommend".
    - Submit button.
    - Display feedback using Alert or Modal.
3. Responsive Grid
    - Use FlatList with numColumns={2}.
    - Display categories in Card style.
    - Each card contains Icon and Text.
    - Add press interaction.
# Additional Exercises
1. Create a multi-step registration form:
    - Screen 1: Personal details.
    - Screen 2: Preferences (checkbox & switch).
    - Screen 3: Summary screen.
    - Use React Navigation.
    - Validate all inputs before moving next.
2. Create a Dynamic Survey App:
    - Fetch questions from JSON array.

    - Add reset functionality.
3. Develop a Product Listing App:
    - Use FlatList with search filter.
    - Implement pull-to-refresh.
    - Add dynamic sorting (price ascending/descending).
